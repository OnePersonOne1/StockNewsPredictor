## 1. 동기 / 가설
EXP-I 의 확신도 보정에서 KOSPI h5 고확신(|score| 상위 1/3) 날 적중률 0.667 을 봤다.
이를 **전체 vs 고확신**으로 IC·이진·롱숏 백테스트 모두에서 정량화 — 신호가 '확신일에
농축'됨을 정량 확증.

## 2. 방법
다년 모델 점수로 각 (index,h) 셀에서 전체 244행과 **고확신(|score| 상위 1/3, n≈81)**
부분집합에 `signal_test.metrics_for` 를 적용(IC, 이진 acc+binom p, 롱숏+순열 p, hit).

## 3. 결과 (전체 → 고확신)

| index·h | 이진acc | IC | 방향 적중률(hit) |
|---|---|---|---|
| **KOSPI h5** | 0.542 → **0.661** | +0.182** → **+0.210*** | 0.549 → **0.667** |
| KOSDAQ h5 | 0.474 → 0.529 | +0.112* → **+0.198*** | 0.488 → 0.506 |
| KOSDAQ h1 | 0.537 → 0.574 | +0.063 → +0.004 | 0.520 → 0.593 |
| KOSPI h1 | 0.575 → 0.585 | +0.120* → +0.070 | 0.553 → 0.549 |

(고확신 n≈81. *p<.10 **p<.05 ***p<.01; IC p값은 셀의 Spearman.)

## 4. 분석
- **h=5 에서 신호가 확신일에 뚜렷이 농축**: KOSPI h5 이진정확도 0.542→**0.661**,
  적중률 0.549→**0.667**, IC 0.182→0.210. KOSDAQ h5 도 IC 0.112→0.198. → EXP-H 의
  전체-평균 IC 는 사실 **고확신 1/3 구간이 견인**한다.
- **h=1 은 농축 효과가 약함/혼재**(KOSPI h1 IC 오히려 하락) — 초단기 점수가 더 잡음.
- **정직한 caveat**: 고확신 부분집합은 n≈81 로 줄어 분산↑ → KOSPI h5 IC 의 p값이
  0.004(전체)→0.060(고확신)으로 *유의수준은 떨어진다*(효과크기는 커지지만 표본이 작아짐).
  즉 "효과는 강해지나 표본이 작아 단정은 신중". **롱숏 백테스트는 고확신에서도 비유의**
  → 통계적 신호이지 거래 alpha 아님(EXP-H 결론 유지).
