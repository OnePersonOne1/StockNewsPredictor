## 1. 동기 / 가설
EXP-H 에서 약하지만 실재하는 양(+) 신호(특히 KOSPI h5, IC≈0.15)를 검출했다. **그 신호가
어디서 오는가?** (a) 모델이 확신할 때 더 맞는가(확신도 보정), (b) 무슨 헤드라인에
주목해 신호를 내는가(attention 내용)를 해석한다.

## 2. 방법
다년 모델로 test 각 일의 `score=P(up)−P(down)`·attention·실제수익을 수집.
- **확신도 보정**: |score| 3분위(Q1<Q2<Q3)별 방향 적중률(sign(score)=sign(ret)).
- **키워드/사례**: 정확히 맞힌 날의 top-α(상위 attention) 헤드라인 빈출 토큰 + 고확신 사례.

## 3. 결과

**확신도 보정 — |score| 분위별 방향 적중률(무작위 0.5):**

| index·h | Q1(저확신) | Q2 | Q3(고확신) |
|---|---|---|---|
| **KOSPI h5** | 0.512 (n82) | 0.469 (n81) | **0.667 (n81)** |
| KOSPI h1 | 0.52 | 0.59 | 0.55 |
| KOSDAQ h1 | 0.48 | 0.49 | 0.60 |
| KOSDAQ h5 | 0.46 | 0.49 | 0.51 |

→ **KOSPI h5 에서 고확신(Q3) 날의 방향 적중률이 66.7%** (평균|score| 0.117), 저확신
(Q1 0.023)의 ~0.5 대비 뚜렷이 높다. KOSDAQ h1 Q3 도 0.60. (KOSDAQ h5 는 평평~0.5 = 약함.)

**KOSPI h5 정확히 맞힌 날의 top-α 헤드라인 빈출 토큰:**
- **상승 맞힘**: 실적·분기·증시·미국·증가·물가·골드만삭스·상향 → **어닝·미국 매크로** 테마.
- **하락 맞힘**: 비트코인·기준금리·수출·한은·뉴욕증시·달러·반도체 → **금리·환율·수출 리스크** 테마.

**고확신 정답 사례(top-α 헤드라인):**
- 2024-07-04 score=+0.17 ret=+0.024 | "美, 6월 새 일자리 줄고…노동시장 냉각"(금리인하 기대)
- 2024-04-26 score=+0.17 ret=+0.029 | "美 신규 실업수당 청구 줄어…전망치 밑돌아"
- 2024-05-31 score=+0.17 ret=+0.025 | "HP PC 매출 2년만에 증가…美 PC시장 반등신호"

## 4. 분석
- **신호는 고확신 날에 집중**: EXP-H 의 IC≈0.15(전체 평균)는 사실 **확신일에 농축**된
  것. KOSPI h5 고확신 1/3 구간에서 ⅔가 방향을 맞힌다 → 신호가 실재하고 *해석 가능*함을
  뒷받침. (단 KOSPI h5 는 엄밀 단조는 아님 — Q2 가 약간 하락; 표본 n≈81.)
- **모델은 잡음이 아니라 시장 본질 테마에 주목**: 어닝(실적/분기)·미국 매크로·금리·환율·
  수출 — 사람이 봐도 지수 방향과 연결되는 내용. 즉 attention 이 학습한 농축(EXP-D·G)이
  *의미 있는* 내용 선택임을 정성적으로 확인.
- **KOSDAQ 는 약함**(보정 평평) — EXP-H 의 KOSPI>KOSDAQ 신호 비대칭과 일관.

## 5. 한계
- 분위 표본 n≈81, **단일 시드(seed 42)** 모델 — 보정 곡선은 ±변동 가능(엄밀 단조 아님).
- 키워드 비교는 **기술통계**(인과 아님), 형태소 분석 없는 어절 빈도.
- KOSPI h5·고확신에 국한된 효과 — 일반화 주의.
