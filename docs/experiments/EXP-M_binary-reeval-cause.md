## 1. 동기 / 가설
EXP-H·I 에서 3-class macro-F1 이 가린 약신호를 **이분법(up/down)·IC** 로 검출했다.
사용자 요청: *"그동안의 모든 실험을 이분법으로 다시 평가해 비교하고, 원인을 찾으라."*
핵심은 **어떤 조건이 방향 신호를 만드는가**이다. 단, EXP-G·L 에서 **단일 시드 IC 가
부호까지 뒤집힘**을 봤으므로, 인과 주장은 반드시 **다중 시드**로 해야 한다(단일 시드
비교는 노이즈에 속는다). 1단계로 **가장 유력한 원인인 '데이터 크기'** 축을 다중 시드로
정량화한다.

## 2. 방법
- 데이터 크기 축 3 케이스: **y2024**(소, 406행)·**y2022**(소·정상기, 406행)·
  **multiyear**(대, 1358행). 각각 RoBERTa 를 **여러 시드**로 재학습하고
  `signal_test` 로 IC(Spearman, score=P(up)−P(down) vs ret_h)·이진 acc 측정.
  - y2024·y2022: 시드 0–4 (5개). multiyear: EXP-H 의 시드 42,0–4 (6개) 재사용.
- 비교 지표는 **IC 의 평균·표준편차·부호 일관성**(신호의 *안정성*이 핵심).

## 3. 결과 (RoBERTa IC, 다중 시드; `figures/signal_datasize.png`)

| case | 규모 | KOSPI h5 IC mean±std | 양수/시드 | KOSDAQ h5 IC | 비고 |
|---|---|---|---|---|---|
| y2024 | 소(406) | **−0.175 ± 0.105** | 0/5 | −0.214 ± 0.147 | 음·노이즈 |
| y2022 | 소(406, 정상) | −0.082 ± **0.304** | 3/5 | −0.134 ± 0.286 | 사실상 무작위 |
| **multiyear** | **대(1358)** | **+0.146 ± 0.026** | **6/6** | +0.068 ± 0.033 | **안정적 양(+)** |

(h=1 도 동일 경향: 소표본 IC 평균 음/0 + 큰 std, 대표본만 안정 양.)

## 4. 분석 — **원인: 데이터 크기 (이분법 렌즈로 재확인)**
1. **소표본은 IC 조차 불안정하다.** y2024·y2022 모두 IC 평균이 0 근처/음수이고
   **표준편차가 ±0.10~0.30** 에 달한다 — 시드마다 부호가 바뀌는 **노이즈**다. 즉
   이분법/IC 라는 더 민감한 렌즈로 봐도 **소표본엔 살릴 방향 신호가 없다**(macro-F1
   붕괴(EXP-A·F)와 동일 현상의 더 깊은 확인).
2. **대표본만 안정적 양(+) 신호.** multiyear 는 IC +0.146±**0.026**(KOSPI h5) 로
   **부호 6/6 일관·분산이 ~10배 작다.** 신호의 *출현*은 곧 **분산의 붕괴**다.
3. **이로써 그동안의 모든 결과가 하나로 통합된다**:
   - EXP-A·B·C 의 예측·attention 붕괴, EXP-D 의 회복, EXP-F 의 정상기 소표본 붕괴,
     EXP-G·L 의 단일 시드 IC 부호 flip — **전부 같은 '데이터 크기 → IC 분산' 현상**.
   - 특히 EXP-L 의 it(−0.135) vs market_it(+0.108) 부호 엇갈림은 *IT 보강 효과가 아니라*
     소·중 표본에서의 IC 분산(이 그래프의 빨강/주황 오차막대)으로 설명된다.
4. **방법론적 함의**: 필터·본문·IT 보강 같은 **2차 요인의 RoBERTa IC 비교는 단일 시드로
   불가**(분산이 효과보다 큼). 안정적으로 측정 가능한 건 *대표본·전체(all)* 구성뿐이며,
   그 위에서만 약신호(+0.07~0.15)가 재현된다. (TF-IDF 는 결정적이라 필터 비교가 신뢰됨 —
   EXP-C·E·G 의 TF-IDF 결론은 유지.)

## 5. 한계
- 1단계는 **데이터 크기 축**에 집중(가장 유력 원인). 필터/본문/IT 축의 다중 시드 IC 는
  미수행(비용) — 단일 시드 signal_metrics 는 각 results 폴더에 있으나 §4 이유로 *기술통계*로만.
- 대표본 IC 도 약함(0.07~0.15, 분산 ~2%)·롱숏 백테스트 비유의(EXP-H) → 거래 alpha 아님.
- 시드 5–6개 — 더 많으면 std 추정 정밀↑.

## 6. 결론 / 다음
- **"이분법으로 다시 봐도 원인은 데이터 크기"**: 소표본 IC 는 부호조차 불안정한 노이즈,
  대표본만 안정적 약신호. 신호 출현 = 분산 붕괴. 모든 이전 실험이 이 한 축으로 설명된다.
- 다음(EXP-M 2단계 후보): 필터/본문/IT 축의 **다중 시드 IC** 로 2차 요인 정밀 비교,
  또는 IT_section 단독(EXP-N).
